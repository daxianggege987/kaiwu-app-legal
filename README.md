# 开悟人生 · 公开文档仓库

本仓库仅包含：

- **`docs/support.html`** — 用户技术支持页面（可托管为 GitHub Pages 或外链）
- **`docs/privacy.html`** — 隐私政策

应用 **源代码未包含在本仓库中**（见根目录 `.gitignore`）。

## 使用前请替换占位内容

在两个 HTML 文件中，将 **`YOUR_EMAIL`**（及 `mailto:YOUR_EMAIL`）替换为您的真实支持邮箱。

## 建议在 GitHub 上启用 Pages

仓库 **Settings ▸ Pages**：选择分支与 `/docs` 根目录发布后，将得到类似：

`https://<username>.github.io/<repo>/support.html`  
`https://<username>.github.io/<repo>/privacy.html`

将上述链接填入 App Store Connect 的「技术支持网址」「隐私政策网址」。
