## 描述（Description，可多段）

**第一段（卖点摘要）**

开悟人生帮助你用「悟道卡牌」与「修行感悟日记」把日常的觉察记录下来：觉醒进度与本机勾选同步，桌面小组件可一眼看到当前摘要与顿悟提示。所有日记与抽卡记录默认保存在你的设备；也可导出 JSON 备份，换机更可自行导入。**重要：卸载 App 可能清空本地数据，请及时备份。**

**功能要点**

• 悟道图鉴：多类卡牌详尽说明，勾选「已达成」累计觉醒进度与境界示意（算法为应用内游戏规则，不构成医疗或心理辅导）。  
• 今日修行：每日随机三张悟道提示，并可查看详情与历史记录（可在设置中清空记录）。  
• 修行日记：撰写、浏览与删除单条心得，字数统计与本机持久化存储。  
• 小组件：在桌面展示境界、觉醒进度及近期悟道摘要（需在系统中添加）。  
• 浅色 / 深色：可跟随系统或在应用内自选（以当前版本为准）。  
• 快捷指令支持：可按需跳转应用或触发应用内脚本（取决于系统捷径配置）。
