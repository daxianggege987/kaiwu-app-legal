本产品为本地内容与修行记录工具；如遇问题可先查阅以下内容。

订阅与付费

  • 订阅由 Apple App Store 计费与续订，可随时在「设置 ▸ Apple ID ▸ 订阅」中管理或取消。
  • 若在应用内加载不到价格，请确认网络正常；开发者账号侧商品已就绪时，仍可稍后再试或使用「刷新订阅信息」。
  • 恢复购买：使用购买时所用的 Apple ID 登录后，在应用「设置 ▸ 订阅与试用」中选择「恢复购买」。

数据、备份与小组件

  • 日记与抽卡记录主要保存在本机；可通过菜单中的「导出 / 导入」使用 JSON 备份文件迁移或备份。
  • 小组件显示摘要信息，需在系统设置中为应用开启桌面小组件并按需选取尺寸。
  • 卸载应用可能删除本地数据；重要内容请务必先导出备份。

常见故障排查

  • iPad 上界面过窄或过宽：请更新至最新版本；若仍为旧版请联系支持邮箱并说明机型与系统版本。
  • 页面打不开或空白：尝试完全退出应用再打开；若持续出现，请联系支持并提供 iOS / iPadOS 版本号。
  • 捷径 / 快捷指令无反应：确认已允许应用相关权限；在应用至少成功打开过一次主界面后再试。
